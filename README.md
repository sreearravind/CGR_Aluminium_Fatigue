# Fatigue Crack Growth Rate of Aluminium Alloys using Fortran

This repository is created to practice Fortran programming through fatigue and fracture mechanics problems. The initial implementation focuses on fatigue crack growth rate prediction using the Paris law model for aluminium alloys.

## Governing Equation

da/dN = C(ΔK)^m

where:
- da/dN = fatigue crack growth rate
- C and m = material constants
- ΔK = stress intensity factor range
